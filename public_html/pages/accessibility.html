Accessibility Statement

Committed to providing an accessible learning environment for all

Our Commitment to Accessibility

Web Accessibility Standards

Math and Language Synergy is committed to ensuring digital accessibility for people with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

We aim to conform to level AA of the Web Content Accessibility Guidelines (WCAG) 2.1. These guidelines explain how to make web content more accessible for people with disabilities.

Accessibility Features

  • Keyboard navigation support
  • Screen reader compatibility
  • Alternative text for images
  • Clear and consistent navigation
  • Responsive design for various devices
  • High contrast mode support
  • Text resizing capabilities

Technical Specifications

This website relies on the following technologies to work with the particular combination of web browser and any assistive technologies or plugins installed on your computer:

  • HTML
  • CSS
  • JavaScript

Feedback & Assistance

We welcome your feedback on the accessibility of Math and Language Synergy. Please let us know if you encounter accessibility barriers:

  • Phone: +27 (0)11 234 5678
  • Email: accessibility@mathlanguagesynergy.edu
  • Visitor address: 123 Learning Lane, Knowledge City, EDU 45678

We try to respond to feedback within 2 business days.

Compatibility

Our website is designed to be compatible with:

  • Recent versions of screen readers
  • Operating system accessibility features
  • Modern web browsers (Chrome, Firefox, Safari, Edge)
  • Mobile devices and tablets

Additional Notes

This statement was created on October 1, 2023 using the W3C Accessibility Statement Generator Tool.

We continuously monitor our website and make accessibility improvements as needed. This statement will be reviewed and updated regularly.
